Ví dụ 2: Chứng minh rằng S = 1 + 3 + 32 + 33 + 34 + ... + 311 chia hết cho 40.

Ta có: S = 1 + 3 + 32 + 33 + 34 + ... + 311
= ( 1 + 3 + 32 + 33) + 34(1 + 3 + 32 + 33) + 38(1 + 3 + 32 + 33)
= (1 + 3 + 32 + 33)( 1 + 34 + 38)
= 40(1 + 34 + 38) ⋮ 40 (Điều phải chứng minh).

Ví dụ 1: Tìm số tự nhiên n để 5 chia hết cho n-1

Giải:

Để 5 chia hết cho n-1 ( n ∈ N ) thì n-1 ∈ U(5).

Ta có: U(5) = {1 ,5 }

  • Với n-1 = 1 => n=2
  • Với n-1 = 5 => n=6

Vậy n ∈ { 2, 6} thõa mãn yêu cầu đề bài.

Ví dụ 2: Tìm số tự nhiên n để n+10 chia hết cho n+5

Giải:

Phương pháp : Ta đưa bài toán về dạng cơ bản như ví dụ 1, triệt tiêu vế trái thành số tự nhiên.

Ta có: n+10 ⋮ n+5
=> (n+10) - (n+5) ⋮ (n+5)
=> 5 ⋮ (n+5)
=> n+5 ∈ U(5)

Để 5 chia hết cho n+5 ( n ∈ N ) thì n-1 ∈ U(5).

Ta có: U(5) = {1 ,5}

  • Với n+5 = 1 (Điều này không xảy ra do n ∈ N)
  • Với n+5 = 5 => n=0

Vậy n ∈ {0} thõa mãn yêu cầu đề bài.
